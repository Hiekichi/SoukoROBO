# SoukoROBO

## 概要
- プログラミング学習用に作成したゲームプログラムです。
- ナニカ問題がありましたら対応させていただきたいと思います。

## 遊び方など
- キーボードのカーソルキー／ゲームパッドの十字キーで移動
- キーボードのEnterキー／ゲームパッドのXボタンでステージやり直し
- キーボードのスペースキー/ゲームパッドのAボタンで次ステージへ進む
- 荷物は押すだけ
